¿Qué documentos necesitas para solicitar un acta de nacimiento?

Requisitos y documentación necesaria.

Información Detallada

La solicitud de un acta de nacimiento requiere una documentación específica para garantizar la autenticidad del trámite.

Documentos generales para el solicitante

  • Identificación oficial vigente con fotografía:
    • Credencial de elector (INE)
    • Pasaporte
    • Cédula profesional
  • Comprobante de domicilio reciente (no mayor a 3 meses):
    • Recibo de luz
    • Recibo de agua
    • Recibo de teléfono
    • Estado de cuenta bancario

Documentos adicionales según el caso

  • Para solicitar acta propia:
    • CURP (opcional pero recomendado)
    • Datos precisos de lugar y fecha de nacimiento
  • Para solicitar acta de un familiar:
    • Documentación que acredite el parentesco
    • Autorización o poder legal (en algunos casos)

Documentos específicos para casos especiales

  • Menores de edad:
    • Identificación de padre, madre o tutor
    • Documentos de representación legal
  • Extranjeros:
    • Pasaporte
    • Documento migratorio vigente
    • Documentos de traducción oficial (si aplica)

Recomendaciones importantes:

  • Verificar la vigencia de todos los documentos
  • Llevar documentos originales
  • Contar con copias legibles de cada documento
  • Consultar requisitos específicos de cada entidad

La documentación puede variar ligeramente según el estado y el tipo de trámite, por lo que se recomienda confirmar los requisitos específicos previamente.
